Designing Delegates Examples
============================

This package contains the source code for the example described in the
"Designing Delegates" article in Qt Quarterly 24.

The "treebox" and "advtreebox" directories contain example code for the
ExpDelegate and AdvExpDelegate classes, respectively. Project files for
building the examples are provided.

About Qt Quarterly
