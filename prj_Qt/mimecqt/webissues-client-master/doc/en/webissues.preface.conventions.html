Used conventions

To facilitate reading this book, the following typographical conventions are used:

Term

First use of a term with its definition.

Name

Name of a user interface element – window, button, label, etc.

File

Name of a file or directory.

Some information has been placed in special boxes:

Tip

Useful information, hint, or recommended practice.

Warning

Important information which requires special attention.

Note

Additional information intended mostly for advanced users.
